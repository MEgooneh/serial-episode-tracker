# Serial Episode Tracker

Chrome extension that adds checkboxes next to video files on directory listing pages so you can track what you've watched.

<img width="1500" height="849" alt="image" src="https://github.com/user-attachments/assets/3f92e8cc-00d4-4e02-890c-371483999a1c" />

## Features

- Checkbox next to each video file link (.mkv, .mp4, .avi, etc.)
- Watched episodes get strikethrough styling
- Progress counter with progress bar
- State persists across sessions

## Install

1. Go to `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this folder
